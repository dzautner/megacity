# UX-068: Grid Snap Toggle for Cursor

## Priority: T1 (Core Polish)
## Effort: Small (0.5 day)
## Source: camera_controls_ux.md -- Road Snapping

## Description
Toggle between free cursor movement and grid-snapped cursor. Grid snap useful for precise road placement and building alignment. Ctrl key currently toggles legacy grid mode.

## Acceptance Criteria
- [ ] Toggle key for grid snap mode
- [ ] In grid snap: cursor snaps to cell centers
- [ ] In free mode: cursor moves freely
- [ ] Visual indicator showing grid snap is active
