# PROG-011: Blueprint Drafting System (Against the Storm inspired)

**Category:** Progression / Roguelite
**Priority:** T4
**Source:** indie_competitors.md -- Against the Storm

## Summary

At each milestone, choose 1 of 3 random building unlocks (blueprint drafting). Forces adaptation to available buildings rather than always using same strategy. Rerolls as limited resource. Creates varied playthroughs.

## Acceptance Criteria

- [ ] Random building selection offered at milestones
- [ ] Player chooses from 3 options
- [ ] Rerolls available but limited
- [ ] Each playthrough has different building set
