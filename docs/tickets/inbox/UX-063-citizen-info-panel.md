# UX-063: Citizen Info Panel (Name, Job, Happiness, Commute)

## Priority: T2 (Depth)
## Effort: Small (1-2 days)
## Source: camera_controls_ux.md -- Section 10.1: Citizen Selection

## Description
When a citizen is selected (clicked), show: name, age, gender, job, happiness, current state, home/work locations. "Follow" button to enter camera follow mode.

## Acceptance Criteria
- [ ] Citizen name, age, gender displayed
- [ ] Job type and workplace shown
- [ ] Happiness with factor breakdown
- [ ] Current state (at home, commuting, at work, shopping)
- [ ] Home and work locations (clickable to jump camera)
- [ ] "Follow" button enters camera follow mode
