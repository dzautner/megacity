# FEAT-002: Road Building UX Improvements

**Category:** Feature / Transportation
**Priority:** T1
**Source:** community_wishlists.md -- Section 1.2 (EXTREMELY HIGH frequency)

## Summary

Better road placement tools, forgiving snap behavior, undo for roads, roundabout builder, interchange templates, Move It equivalent (move placed roads/buildings), road anarchy (remove slope limits).

## Details

- Improved grid snapping with more forgiving tolerance
- Undo/redo for road placement and demolition
- Built-in roundabout builder (select size, auto-generate)
- Pre-built interchange templates (cloverleaf, turbine, stack)
- Move already-placed roads without demolishing
- Override slope angle limitations
- Road hierarchy awareness (visual guidance for local-collector-arterial-highway)

## Acceptance Criteria

- [ ] Road snap behavior improved
- [ ] Undo/redo functional for road operations
- [ ] Roundabout builder tool available
- [ ] At least 3 interchange templates available
- [ ] Move roads/buildings without demolish
