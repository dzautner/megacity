# FEAT-040: Technology Tree and Era Progression

**Category:** Feature / Gameplay
**Priority:** T3
**Source:** community_wishlists.md -- Section 12.2, 19

## Summary

Advance through historical or technological eras unlocking new capabilities. Tech tree: internet, fiber optic, 5G, smart grid. Start in different eras (village, town, established city). Period-appropriate technologies and challenges.

## Details

- Era progression: settlement -> town -> city -> metropolis -> megacity
- Each era unlocks new buildings, services, transit types, policies
- Optional historical start (1900, 1950, 2000, modern)
- Smart city technologies: autonomous vehicles, smart traffic, digital infrastructure
- Tech research funded by education/university system

## Acceptance Criteria

- [ ] At least 3 eras with distinct available tech
- [ ] Unlock buildings/services as era progresses
- [ ] Era transition triggered by milestones
- [ ] Smart city tech available in late eras
