# FEAT-044: Props, Decorations, and Seasonal Elements

**Category:** Feature / Player Expression
**Priority:** T3
**Source:** game_design_mechanics.md -- Section 3.3

## Summary

Decorative objects (statues, fountains, murals, clock towers, arches). Street-level details (food carts, newspaper stands, bus shelters). Seasonal decorations (holiday lights, autumn leaves). Unlockable props earned through achievements.

## Acceptance Criteria

- [ ] Decorative prop placement tool
- [ ] At least 20 prop types
- [ ] Seasonal decoration auto-placement
- [ ] Props unlockable through achievements
