# FEAT-064: Stormwater Management and Flooding

**Category:** Feature / Infrastructure
**Priority:** T3
**Source:** master_architecture.md, community_wishlists.md

## Summary

Stormwater drainage network (separate or combined with sewage). Flooding from inadequate capacity during heavy rain. Green infrastructure options (rain gardens, green roofs, bioswales, permeable surfaces). Retention ponds and detention basins.

## Acceptance Criteria

- [ ] Stormwater drainage modeled
- [ ] Flooding when capacity exceeded
- [ ] Green infrastructure options available
- [ ] Flood plain mapping visible
