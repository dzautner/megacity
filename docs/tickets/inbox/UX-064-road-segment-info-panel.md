# UX-064: Road Segment Info Panel

## Priority: T2 (Depth)
## Effort: Small (1-2 days)
## Source: camera_controls_ux.md -- Section 10.1: Road Selection

## Description
When a road segment is selected, show: road type, length, traffic volume, congestion level, connected intersections, maintenance cost.

## Acceptance Criteria
- [ ] Road type and width shown
- [ ] Segment length in meters
- [ ] Current traffic volume
- [ ] Congestion level (LOS grade A-F)
- [ ] Connected intersection nodes
- [ ] Monthly maintenance cost
