# FEAT-024: Customizable Parks System

**Category:** Feature / Buildings
**Priority:** T2
**Source:** community_wishlists.md -- Section 7.3 (HIGH frequency)

## Summary

Player-designed parks with modular components (paths, benches, fountains, playgrounds, bandstands, gardens, sports courts). Not just pre-made plop-and-play. Real-world landmarks as placeable special buildings. Stadium/arena events with game-day traffic.

## Acceptance Criteria

- [ ] Park designer with modular components
- [ ] Parks provide land value / happiness / recreation bonuses
- [ ] Stadium events generate traffic spikes
- [ ] Landmark buildings placeable
