# FEAT-051: Progressive Tutorial System

**Category:** Feature / UI
**Priority:** T2
**Source:** game_design_mechanics.md -- Section 5.4

## Summary

Progressive disclosure: teach one mechanic at a time, in context. Guided overlays, advisor-driven tutorials, interactive challenges, reference library, tooltip hints, contextual tips. Never front-load all tutorials at start.

## Acceptance Criteria

- [ ] Tutorials trigger contextually (first road -> road tutorial)
- [ ] Guided overlay highlighting UI elements
- [ ] Interactive tutorial challenges
- [ ] Reference library accessible from help menu
- [ ] Tooltips on all UI elements
