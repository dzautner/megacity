# FEAT-094: Cemetery and Death Care Services

**Category:** Feature / Services
**Priority:** T2
**Source:** master_architecture.md

## Summary

Death care as city service. Cemeteries and crematoriums. Capacity limits (cemeteries fill up). Hearse dispatch to pick up deceased. Cemetery land value impact on neighbors. Memorial parks dual-use (cemetery + park).

## Acceptance Criteria

- [ ] Cemetery/crematorium buildings
- [ ] Hearse dispatch from nearest facility
- [ ] Cemetery capacity tracking
- [ ] Memorial park option (cemetery + recreation)
