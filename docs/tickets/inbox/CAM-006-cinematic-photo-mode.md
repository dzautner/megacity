# CAM-006: Cinematic / Photo Mode

## Priority: T4 (Polish)
## Effort: Medium (3-5 days)
## Source: camera_controls_ux.md -- Section 6: Photo Mode, master_architecture.md T4

## Description
Hide all UI, unlock camera constraints, enable DOF (depth of field), color grading, and time-of-day lock. For screenshots and videos.

## Acceptance Criteria
- [ ] Toggle key (e.g., F10) enters photo mode
- [ ] All UI hidden
- [ ] Camera pitch/distance limits removed
- [ ] DOF controls: focal distance, aperture
- [ ] Color grading: brightness, contrast, saturation, temperature
- [ ] Time-of-day lock slider
- [ ] Screenshot button (F12) saves high-res image
