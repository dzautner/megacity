# FEAT-019: Multi-Depth Underground Building

**Category:** Feature / Infrastructure
**Priority:** T3
**Source:** community_wishlists.md -- Section 5.3

## Summary

Build at different depths underground. Utility tunnels (shared corridors for pipes/cables). Subway construction methods (cut-and-cover vs TBM with different costs/disruption). Underground parking connected to surface.

## Acceptance Criteria

- [ ] Multiple underground depth levels
- [ ] Utility tunnels constructible
- [ ] Underground view with depth selection
- [ ] Underground parking functional
