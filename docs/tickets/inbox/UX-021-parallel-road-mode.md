# UX-021: Parallel Road Drawing Mode

## Priority: T2 (Depth)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 12.2: Road Drawing Modes

## Description
After placing a road, hold Shift to place a parallel road at configurable offset. Essential for boulevard and highway construction with matched pairs.

## Acceptance Criteria
- [ ] Shift+placement creates parallel road
- [ ] Configurable offset distance
- [ ] Preview shows both original and parallel road
- [ ] Works with both straight and curve modes
