# UX-041: Blueprint / Template System

## Priority: T4 (Polish)
## Effort: Large (1-2 weeks)
## Source: camera_controls_ux.md -- Blueprint System, master_architecture.md T4

## Description
Save and load reusable city layouts (road patterns, zone configurations, service placements). Select area, save as blueprint, stamp down elsewhere.

## Acceptance Criteria
- [ ] Select-area tool for defining blueprint region
- [ ] Save blueprint to file (roads, zones, services within region)
- [ ] Blueprint library panel
- [ ] Stamp blueprint at cursor location
- [ ] Blueprint preview during placement (transparent overlay)
- [ ] Rotation support (0, 90, 180, 270 degrees)
- [ ] Shareable blueprint files
