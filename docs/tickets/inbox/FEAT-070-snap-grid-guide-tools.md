# FEAT-070: Snap, Grid, and Guide Line Tools

**Category:** Feature / Tools
**Priority:** T2
**Source:** game_design_mechanics.md -- Section 7.6

## Summary

Snap-to-grid toggle. Free-form mode (disable grid). Angle snapping (15/30/45/90 degrees). Visual guide lines for alignment with nearby buildings/roads. Height snapping to terrain. CS2-style visual helpers for parallel roads and curve alignment.

## Acceptance Criteria

- [ ] Grid snap toggle
- [ ] Angle snapping for roads
- [ ] Visual guide lines during placement
- [ ] Height snap to terrain
