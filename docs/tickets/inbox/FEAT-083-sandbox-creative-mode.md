# FEAT-083: Sandbox / Creative Mode

**Category:** Feature / Game Modes
**Priority:** T2
**Source:** game_design_mechanics.md -- Section 2.4

## Summary

Dedicated sandbox mode: unlimited money, all buildings unlocked, no disasters, pure creative mode. Separate from normal/hard/survival modes. Clearly labeled. Can toggle individual constraints on/off (e.g., unlimited money but disasters enabled).

## Acceptance Criteria

- [ ] Sandbox mode selectable
- [ ] Unlimited money toggle
- [ ] All buildings unlocked toggle
- [ ] Disaster toggle
- [ ] Individual constraint toggles
