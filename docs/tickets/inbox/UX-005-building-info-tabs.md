# UX-005: Tabbed Building Info Panel

## Priority: T2 (Depth)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 13.2: Building Info Panel

## Description
Building info panel has too much information in a flat list. Organize into tabs: Overview, Services, Economy, Residents/Workers, Environment.

## Acceptance Criteria
- [ ] Tab bar at top of building info panel
- [ ] Overview tab: type, level, occupancy, happiness (one-glance summary)
- [ ] Services tab: coverage and quality for each service type
- [ ] Economy tab: land value, rent, property value, taxes
- [ ] Residents tab: list of citizens (clickable to follow)
- [ ] Environment tab: pollution, noise, green space
