# UX-015: Tool Cost Tooltips (Maintenance, Radius, Capacity)

## Priority: T1 (Core Polish)
## Effort: Small (0.5 day)
## Source: camera_controls_ux.md -- Section 11.2: Tool Activation UX

## Description
