# PROG-001: Production Chain System (Workers & Resources inspired)

**Category:** Progression / Economy
**Priority:** T3
**Source:** indie_competitors.md -- Workers & Resources: Soviet Republic

## Summary

Multi-step production chains from raw materials to finished goods. Each commodity has specific storage types. Factories halt when output is full (backpressure). 10-30+ distinct commodities. Logistics vehicles specific per commodity class.

## Acceptance Criteria

- [ ] At least 10 commodities with extraction-to-finished chains
- [ ] Backpressure when storage full
- [ ] Specific storage types per commodity
- [ ] Visual commodity flow on transport network
