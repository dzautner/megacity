# PROG-004: Tiered Population Needs (Anno 1800 inspired)

**Category:** Progression / Citizens
**Priority:** T2
**Source:** indie_competitors.md -- Anno 1800

## Summary

Citizens progress through tiers with escalating needs. Each tier unlocks new buildings/production chains. Tier advancement requires fulfilling all needs of current tier. Needs hierarchy from basic (food, shelter) to luxury (entertainment, culture, education).

## Details

- Tier 1: Basic needs (food, water, housing)
- Tier 2: Comfort needs (electricity, heating, clothing)
- Tier 3: Community needs (schools, healthcare, parks)
- Tier 4: Cultural needs (entertainment, culture, dining)
- Tier 5: Aspirational needs (university, nightlife, luxury goods)

## Acceptance Criteria

- [ ] Citizens have tiered needs
- [ ] Higher tiers unlock more buildings
- [ ] Need fulfillment drives tier advancement
- [ ] Different tier citizens have different economic contributions
