# FEAT-073: Weather Forecast UI

**Category:** Feature / UI
**Priority:** T3
**Source:** game_design_mechanics.md -- Section 6.6

## Summary

3-day weather forecast displayed in UI. Affects player planning (prepare for storms, schedule construction, manage power). Forecast accuracy degrades further out (day 1: reliable, day 3: uncertain).

## Acceptance Criteria

- [ ] 3-day forecast in UI
- [ ] Forecast affects player decisions
- [ ] Accuracy decreases for further days
- [ ] Weather icons for each forecast day
