# FEAT-007: Mixed-Use Zoning

**Category:** Feature / Zoning
**Priority:** T1
**Source:** community_wishlists.md -- Section 2.1 (EXTREMELY HIGH frequency)

## Summary

Mixed-use buildings with commercial ground floor and residential above. The single most requested feature from urban planners. Vertical zoning (different uses on different floors). Noise-compatible mixing (residential near quiet commercial but not loud).

## Details

- New zone type: Mixed-Use (residential + commercial in same building)
- Vertical zoning: ground floor commercial, upper floors residential
- Noise compatibility rules (cafes ok, nightclubs not ok near residential)
- More granular zone subtypes (light commercial, heavy industrial, live-work)

## Acceptance Criteria

- [ ] Mixed-use zone type available
- [ ] Buildings spawn with commercial ground floor + residential upper
- [ ] Noise compatibility affects which uses can mix
- [ ] Citizens live and shop in same building/block
