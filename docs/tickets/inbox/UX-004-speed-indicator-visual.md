# UX-004: Enhanced Speed Indicator Visual

## Priority: T1 (Core Polish)
## Effort: Small (0.5 day)
## Source: camera_controls_ux.md -- Section 13.1: HUD

## Description
Speed buttons (pause, 1x, 2x, 4x) are not visually prominent. Add color coding: paused = red dot, 1x = green, 2x = yellow, 4x = orange. Active speed highlighted.

## Acceptance Criteria
- [ ] Color-coded speed indicator dots
- [ ] Active speed button clearly highlighted
- [ ] Colors: red (paused), green (1x), yellow (2x), orange (4x)
