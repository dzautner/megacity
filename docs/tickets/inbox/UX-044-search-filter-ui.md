# UX-044: Search/Filter for Buildings and Citizens

## Priority: T3 (Differentiation)
## Effort: Medium (3-5 days)
## Source: camera_controls_ux.md -- Search/Filter

## Description
Search bar that filters buildings by type, level, or status. Filter citizens by name, occupation, happiness, or home/work location. Results shown in scrollable list; clicking jumps camera.

## Acceptance Criteria
- [ ] Search bar accessible via keybind (Ctrl+F)
- [ ] Search buildings by type, zone, level, status (abandoned, under construction)
- [ ] Search citizens by name, age, occupation
- [ ] Results in scrollable list with brief info
- [ ] Click result to jump camera to entity
