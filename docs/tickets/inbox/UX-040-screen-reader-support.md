# UX-040: Screen Reader Support (Basic Accessibility)

## Priority: T4 (Polish)
## Effort: Large (1-2 weeks)
## Source: camera_controls_ux.md -- Section 16: Accessibility

## Description
Basic screen reader support for menus, tool selection, and information panels. Announce selected building info, tool changes, and notifications.

## Acceptance Criteria
- [ ] ARIA-like labels on UI elements (via egui accessibility features)
- [ ] Tool selection announced
- [ ] Building info readable
- [ ] Notification text announced
- [ ] Menu navigation via keyboard
