# FEAT-013: Persistent Citizen Lives and Daily Schedules

**Category:** Feature / Citizens
**Priority:** T1
**Source:** community_wishlists.md -- Section 4.1 (VERY HIGH frequency)

## Summary

Citizens have fixed homes, workplaces, and routines. Daily schedules (wake, commute, work, shop, recreate, sleep). Citizen needs hierarchy (food, shelter, safety, entertainment, education). Personality traits affecting behavior. Life stages (born, educated, work, retire, die) without death waves.

## Details

- Fixed home-work assignment (already exists)
- Daily schedule with time-based state machine (partially exists)
- Individual needs hierarchy driving behavior
- Personality traits affecting location preferences
- Household simulation (families, children, lifecycle)
- Death wave prevention (staggered aging)

## Acceptance Criteria

- [ ] Citizens have persistent daily routines
- [ ] Needs hierarchy affects citizen behavior
- [ ] Life events (birth, education, marriage, retirement, death) functional
- [ ] No death waves from synchronized aging
