# UX-051: F12 Screenshot

## Priority: T2 (Depth)
## Effort: Small (1 day)
## Source: camera_controls_ux.md -- Section 14.2: Keybindings

## Description
F12 captures a screenshot of the current view. Saves as PNG to a screenshots directory. Shows brief confirmation message.

## Acceptance Criteria
- [ ] F12 captures current frame to PNG
- [ ] Saved to `screenshots/` directory with timestamp filename
- [ ] Status message: "Screenshot saved"
- [ ] Optional: hide UI before capture (photo mode)
