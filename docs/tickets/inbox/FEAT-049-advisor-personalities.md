# FEAT-049: Multiple Advisor Personalities

**Category:** Feature / UI
**Priority:** T3
**Source:** game_design_mechanics.md -- Section 5.2

## Summary

Multiple advisor types: City Planner, Finance Minister, Transportation Chief, Environmental Officer, Fire/Police Chief, Education Director. Advisors are proactive (warn before crisis), reactive (explain failures), contextual (relevant to current action), dismissible, and configurable. Advisors can disagree, making player the tiebreaker.

## Acceptance Criteria

- [ ] At least 4 advisor types
- [ ] Proactive warnings before problems
- [ ] Reactive explanations after failures
- [ ] Conflicting advisor opinions on tradeoff decisions
