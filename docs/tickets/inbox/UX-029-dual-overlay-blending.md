# UX-029: Dual Overlay Blending / Split-Screen

## Priority: T3 (Differentiation)
## Effort: Medium (3-5 days)
## Source: camera_controls_ux.md -- Section 9.7: Multiple Overlay Blending

## Description
Allow two overlays simultaneously. Primary: full color ramp. Secondary: hatching pattern (diagonal lines, density = value). Alternative: split-screen with draggable divider.

## Acceptance Criteria
- [ ] Primary overlay rendered as normal color ramp
- [ ] Secondary overlay rendered as hatching pattern
- [ ] OR split-screen mode with draggable vertical divider
- [ ] Both overlay names shown in legend
- [ ] Toggle between single and dual overlay modes
