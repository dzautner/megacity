# UX-034: Tab/Shift+Tab Overlay Cycling

## Priority: T1 (Core Polish)
## Effort: Small (0.5 day)
## Source: camera_controls_ux.md -- Section 14.2: Keybindings

## Description
Tab key cycles through overlay modes (None -> Traffic -> Pollution -> ...). Shift+Tab cycles in reverse. Frees letter keys for other shortcuts.

## Acceptance Criteria
- [ ] Tab cycles overlays forward
- [ ] Shift+Tab cycles overlays backward
- [ ] Current overlay shown in status bar
- [ ] "None" overlay in cycle to disable
