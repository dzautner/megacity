# UX-037: Controller/Gamepad Support

## Priority: T4 (Polish)
## Effort: Large (1-2 weeks)
## Source: camera_controls_ux.md -- Section 15: Controller Support, master_architecture.md T4

## Description
Full gamepad input mapping: left stick = cursor, right stick = camera, triggers = zoom, bumpers = rotate, A = place, B = cancel, X = tool wheel (radial), Y = info, D-pad = speed.

## Acceptance Criteria
- [ ] Left stick moves cursor on ground plane
- [ ] Right stick pans camera
- [ ] R3 toggles camera pan/orbit
- [ ] LT/RT zoom out/in
- [ ] LB/RB rotate 90 degrees
- [ ] A button = primary action
- [ ] B button = cancel
- [ ] X button = radial tool wheel
- [ ] Y button = info panel
- [ ] D-pad up/down = speed control
- [ ] Works with Xbox and PlayStation controllers
