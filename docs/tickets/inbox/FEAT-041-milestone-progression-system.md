# FEAT-041: Milestone Progression and Development Trees

**Category:** Feature / Progression
**Priority:** T1
**Source:** game_design_mechanics.md -- Section 1.1, 1.2

## Summary

Two-layer progression: milestones gate services/tools, development trees enable specialization. 20 milestones from "Tiny Village" to "Megapolis". XP earned both actively (building) and passively (population/happiness). Development trees per service category (fire, police, education, healthcare, transport) with tiers: Basic -> Advanced -> Specialized -> Landmark.

## Acceptance Criteria

- [ ] 20+ milestones with progressive unlocks
- [ ] Active + passive XP earning
- [ ] Per-service development trees
- [ ] Specialization paths within each tree
- [ ] Mixed milestone criteria (pop + happiness + services)
