# FEAT-029: Regional Multi-City Play

**Category:** Feature / Regional
**Priority:** T4
**Source:** community_wishlists.md -- Section 11.1 (VERY HIGH frequency)

## Summary

Multiple connected cities in a region that trade, share commuters, and cooperate/compete. Regional transit (commuter rail, inter-city buses). Shared infrastructure (airports, power plants serving multiple cities). Inter-city competition for businesses and residents.

## Acceptance Criteria

- [ ] Multiple cities in shared region
- [ ] Trade between cities
- [ ] Commuter flows between cities
- [ ] Shared infrastructure options
