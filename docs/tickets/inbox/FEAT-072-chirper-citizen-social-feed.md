# FEAT-072: Citizen Social Feed (Chirper Style)

**Category:** Feature / UI
**Priority:** T3
**Source:** game_design_mechanics.md -- Section 10.5

## Summary

Citizens post messages about city conditions reflecting real simulation data. Mix of complaints, praise, humor. Clickable (jump to relevant location). Filterable (complaints only, praise only). Can be minimized or disabled.

## Acceptance Criteria

- [ ] Social feed panel with citizen messages
- [ ] Messages reflect actual simulation state
- [ ] Click to jump to relevant location
- [ ] Filterable by category
- [ ] Minimizable/disableable
