# FEAT-098: Public-Private Partnerships

**Category:** Feature / Economy
**Priority:** T3
**Source:** community_wishlists.md -- Section 10.2

## Summary

Government partners with private developers on projects. Private investment in public infrastructure (hospitals, transit) in exchange for operating rights. TIF (Tax Increment Financing) districts. Developer proposals for zoned areas.

## Acceptance Criteria

- [ ] Private development proposals for zoned areas
- [ ] TIF district designation
- [ ] PPP funding option for large projects
- [ ] Developer builds without city funding (but city loses control)
