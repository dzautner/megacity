# PROG-012: Cornerstone / Perk System (Against the Storm inspired)

**Category:** Progression / Roguelite
**Priority:** T4
**Source:** indie_competitors.md -- Against the Storm

## Summary

Periodic milestone perks that modify gameplay (cornerstones). Legendary, epic, uncommon tiers. Some perks radically alter strategy. Applied at fixed intervals (e.g., every 2 years of game time). Stack for increasingly unique builds.

## Acceptance Criteria

- [ ] Perks offered at regular intervals
- [ ] Tiered rarity system
- [ ] Perks modify gameplay meaningfully
- [ ] Perks stack for compound effects
