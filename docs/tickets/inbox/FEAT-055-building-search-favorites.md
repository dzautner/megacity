# FEAT-055: Building Search, Favorites, and Quick Access

**Category:** Feature / QoL
**Priority:** T2
**Source:** game_design_mechanics.md -- Section 7.7

## Summary

Building search by name/category. Filter by cost, category, era, service type, size, unlock status. Recently used (last 10 buildings). Favorites toolbar. Quick-access customizable hotbar. Keyboard shortcuts fully remappable.

## Acceptance Criteria

- [ ] Search bar for buildings by name
- [ ] Filter by category/cost/size
- [ ] Recently used quick access
- [ ] Favorites star system
- [ ] Customizable hotbar
