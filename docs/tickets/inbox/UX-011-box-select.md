# UX-011: Box Selection (Drag to Select Multiple)

## Priority: T2 (Depth)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 10.3: Box Selection

## Description
Shift+Left-Click drag to select multiple entities in a rectangle. Essential for batch bulldoze, batch zone, batch road upgrade. Translucent colored rectangle on ground plane.

## Acceptance Criteria
- [ ] Shift+Left-Click initiates box select
- [ ] Translucent rectangle rendered on ground during drag
- [ ] On release, all entities in rectangle selected
- [ ] Works with bulldoze, zone, and upgrade tools
- [ ] Selected entity count shown in status bar
