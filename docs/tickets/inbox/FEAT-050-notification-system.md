# FEAT-050: Tiered Notification System

**Category:** Feature / UI
**Priority:** T2
**Source:** game_design_mechanics.md -- Section 5.3

## Summary

Four notification tiers: Critical (red, requires attention), Warning (yellow, important not urgent), Info (blue, informational), Success (green, celebratory). Non-blocking sidebar placement. Clickable (jump to location). Scrollable history with timestamps. Filters by category/severity.

## Acceptance Criteria

- [ ] Four notification tiers with distinct styling
- [ ] Non-blocking sidebar placement
- [ ] Click to jump to relevant location
- [ ] Notification log with history
- [ ] Filterable by category
