# FEAT-026: Social Class and Homelessness Simulation

**Category:** Feature / Social
**Priority:** T3
**Source:** community_wishlists.md -- Section 9.1, 9.2

## Summary

Distinct low/middle/high income groups with different needs. Homelessness from housing unaffordability (visible encampments). Gentrification mechanics. Welfare programs (food assistance, job training, childcare). Wealth visualization overlay. Social mobility based on education and jobs.

## Acceptance Criteria

- [ ] Three income tiers with different needs
- [ ] Homelessness emerges from affordability failure
- [ ] Welfare program options
- [ ] Wealth distribution overlay
- [ ] Social mobility functional
