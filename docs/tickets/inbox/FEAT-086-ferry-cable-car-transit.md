# FEAT-086: Ferry, Cable Car, and Monorail Transit

**Category:** Feature / Transportation
**Priority:** T3
**Source:** community_wishlists.md -- Section 1.3

## Summary

Water-based transit (car ferries, passenger ferries). Cable cars/gondolas for hilly terrain or over water. Monorail for late-game futuristic transit. Each type serves specific geography or density.

## Acceptance Criteria

- [ ] Ferry routes across water bodies
- [ ] Cable car/gondola lines
- [ ] Each transit type serves distinct purpose
- [ ] Station placement and routing tools
