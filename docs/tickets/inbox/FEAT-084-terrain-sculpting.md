# FEAT-084: Terrain Sculpting Tools

**Category:** Feature / Terrain
**Priority:** T3
**Source:** community_wishlists.md -- Section 14.2

## Summary

Raise, lower, flatten, smooth terrain during gameplay. Create water features (lakes, canals). Terrain affects building placement and road construction costs. Slope-based buildability restrictions.

## Acceptance Criteria

- [ ] Terrain raise/lower/flatten/smooth tools
- [ ] Canal digging for water features
- [ ] Slope affects road construction cost
- [ ] Steep slopes restrict building placement
