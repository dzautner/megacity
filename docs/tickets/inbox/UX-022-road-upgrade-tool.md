# UX-022: In-Place Road Upgrade Tool

## Priority: T2 (Depth)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 12.3: Road Upgrade UX

## Description
Click existing road segment with upgrade tool to show upgrade/downgrade options popup. Apply in-place without bulldozing. Show cost difference.

## Acceptance Criteria
- [ ] Road upgrade tool in toolbar
- [ ] Click road shows available upgrades/downgrades
- [ ] Cost = upgrade cost minus salvage value
- [ ] In-place upgrade without traffic disruption
- [ ] Road maintains connections during upgrade
