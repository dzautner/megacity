# FEAT-067: Copy/Paste/Move City Areas

**Category:** Feature / Tools
**Priority:** T3
**Source:** game_design_mechanics.md -- Section 7.3

## Summary

Area selection tool (rectangle or lasso). Copy captures buildings, roads, zones, props. Paste with rotation and mirror options. Move tool: pick up and relocate sections (with cost). Foundation for blueprints.

## Acceptance Criteria

- [ ] Area selection tool
- [ ] Copy/paste with rotation/mirror
- [ ] Move tool for relocating sections
- [ ] Cost shown before confirming move
