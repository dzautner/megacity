# FEAT-003: Modular Road Designer

**Category:** Feature / Transportation
**Priority:** T2
**Source:** community_wishlists.md -- Section 1.2

## Summary

Mix and match lane components: car lanes, bus lanes, bike lanes, parking, medians, trees, sidewalks. Full per-road customization rather than fixed road types.

## Acceptance Criteria

- [ ] Lane components selectable per road
- [ ] Bus/bike lane types available
- [ ] Median and tree options
- [ ] Custom roads save as templates
