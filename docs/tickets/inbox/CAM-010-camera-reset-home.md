# CAM-010: Camera Reset to Default Position (Home Key)

## Priority: T1 (Core Polish)
## Effort: Small (0.5 day)
## Source: camera_controls_ux.md -- Section 14.2: Keybindings

## Description
Home key resets camera to default bird's-eye position: center of map, 45-degree pitch, north-facing yaw, medium zoom. Smooth transition.

## Acceptance Criteria
- [ ] Home key triggers camera reset
- [ ] Smooth transition to default position
- [ ] Default: center of map, 45deg pitch, 0 yaw, medium zoom
