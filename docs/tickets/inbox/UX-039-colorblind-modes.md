# UX-039: Colorblind Accessibility Modes

## Priority: T4 (Polish)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 16: Accessibility, master_architecture.md T4

## Description
Provide color palette alternatives for protanopia, deuteranopia, and tritanopia. Affect all overlay colors, zone colors, traffic indicators, and UI elements.

## Acceptance Criteria
- [ ] Three colorblind modes: protanopia, deuteranopia, tritanopia
- [ ] Settings UI toggle for colorblind mode
- [ ] Overlay color ramps adjusted per mode
- [ ] Zone colors adjusted per mode
- [ ] Traffic red/green indicators use pattern alternatives
