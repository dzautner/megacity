# FEAT-004: Public Transit System

**Category:** Feature / Transportation
**Priority:** T2
**Source:** community_wishlists.md -- Section 1.3 (VERY HIGH frequency)

## Summary

Full public transit system: buses, trams, metros, trains, ferries. Hierarchical transit networks with transfer hubs. Transit ridership driven by rational mode choice (time, cost, convenience). Line profitability and ridership tracking.

## Details

- Transit types: bus, tram/streetcar, metro/subway, train, ferry, cable car
- Hierarchical networks: buses feed trams, trams feed metros, metros feed trains
- Transfer hubs where lines connect
- Mode choice model: citizens compare transit vs car (time, cost, convenience, parking)
- Line management: route drawing, frequency setting, vehicle allocation
- Profitability tracking per line
- Express/local service on same track

## Acceptance Criteria

- [ ] At least 3 transit types (bus, metro, train)
- [ ] Transfer hub support
- [ ] Rational mode choice model
- [ ] Line management UI
- [ ] Ridership tracking
