# UX-061: Progressive Disclosure in Info Panels

## Priority: T2 (Depth)
## Effort: Small (1-2 days)
## Source: camera_controls_ux.md -- Section 13.2: Progressive Disclosure

## Description
Info panels show most important info first (type, level, happiness), with expandable sections for details. Do not dump 30 stats in a flat list.

## Acceptance Criteria
- [ ] Top section: type, level, occupancy, happiness (always visible)
- [ ] Expandable sections for: services, environment, economy, residents
- [ ] Sections remember expanded/collapsed state per session
- [ ] Clean visual hierarchy with headers
