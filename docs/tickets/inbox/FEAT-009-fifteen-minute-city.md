# FEAT-009: 15-Minute City Design Tools

**Category:** Feature / Zoning
**Priority:** T3
**Source:** community_wishlists.md -- Section 2.3

## Summary

All essential services within 15-minute walk/bike of every resident. Walkability/accessibility scoring overlay. Transit-oriented development incentives (higher density around transit). Tools to differentiate suburban sprawl from dense urban cores.

## Acceptance Criteria

- [ ] 15-minute walkability overlay showing service access
- [ ] Transit-oriented development bonus (auto-higher density near transit)
- [ ] Accessibility scoring per neighborhood
