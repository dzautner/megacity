# FEAT-030: Planning Mode (Preview Before Building)

**Category:** Feature / Tools
**Priority:** T2
**Source:** community_wishlists.md -- Section 13.1 (VERY HIGH frequency)

## Summary

Draw future roads/zones without placing them. Preview mode showing planned layout before committing budget. Blueprint/template system to save and reuse road patterns and district designs.

## Details

- Planning layer: draw roads, zones, buildings as plans (different visual)
- Plans show cost estimate before committing
- One-click to commit plan to construction
- Blueprint system: save layout as template, stamp down in other locations
- Template sharing (workshop integration)

## Acceptance Criteria

- [ ] Planning mode toggle in toolbar
- [ ] Plans visible as translucent/dashed overlay
- [ ] Cost estimate shown for plans
- [ ] One-click commit to construction
- [ ] Blueprint save/load functional
