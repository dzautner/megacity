# CAM-012: Focus Camera on Selected Entity (F Key)

## Priority: T1 (Core Polish)
## Effort: Small (0.5 day)
## Source: camera_controls_ux.md -- Section 14.2: Keybindings

## Description
When an entity is selected (building, citizen, road), pressing F centers the camera on it with smooth transition. Standard "focus on selection" behavior.

## Acceptance Criteria
- [ ] F key centers camera on selected entity
- [ ] Smooth transition to new focus point
- [ ] Appropriate zoom level for entity type
- [ ] No-op if nothing is selected
