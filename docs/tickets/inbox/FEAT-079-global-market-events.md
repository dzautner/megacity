# FEAT-079: Global Market Events

**Category:** Feature / Economy
**Priority:** T3
**Source:** community_wishlists.md -- Section 11.2

## Summary

World market changes affecting city: oil price shocks, trade wars, global recession, commodity booms. Import/export prices fluctuate based on global events. City vulnerability depends on trade dependency.

## Acceptance Criteria

- [ ] Global events affect import/export prices
- [ ] City trade dependency creates vulnerability
- [ ] Event notifications with economic impact
- [ ] Player can hedge through diversification
