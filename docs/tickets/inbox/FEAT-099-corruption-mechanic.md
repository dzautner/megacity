# FEAT-099: Corruption Mechanic

**Category:** Feature / Governance
**Priority:** T4
**Source:** community_wishlists.md -- Section 10.2, indie_competitors.md -- Tropico

## Summary

Officials skimming budgets, bribes affecting development decisions. Corruption increases with city size and budget. Anti-corruption measures (oversight, transparency, investigation). Corruption diverts funds, reduces service quality, creates political scandals.

## Acceptance Criteria

- [ ] Corruption risk scales with city size
- [ ] Budget leakage from corruption
- [ ] Anti-corruption policy options
- [ ] Corruption scandals as events
