# FEAT-077: Tiered Healthcare System

**Category:** Feature / Services
**Priority:** T2
**Source:** community_wishlists.md -- Section 4.4

## Summary

Clinics for basic care, hospitals for serious illness, specialists for advanced treatment. Disease/epidemic simulation based on density and sanitation. Mental health effects from noise, pollution, commute time. Ambulance dispatching from nearest station.

## Acceptance Criteria

- [ ] Clinic -> hospital -> specialist tier
- [ ] Disease outbreaks from poor sanitation/density
- [ ] Mental health effects tracked
- [ ] Ambulance dispatch from nearest facility
