# MOD-017: Mod Manager UI

## Priority: T5 (Stretch)
## Effort: Medium (3-5 days)
## Source: modding_architecture.md -- Mod Management, master_architecture.md T5

## Description
In-game UI for managing installed mods: enable/disable, reorder, view details, check for conflicts, view dependencies.

## Acceptance Criteria
- [ ] List of installed mods with enable/disable toggles
- [ ] Drag-to-reorder for load priority
- [ ] Mod details panel: name, author, version, description, dependencies
- [ ] Conflict warnings displayed inline
- [ ] "Apply and restart" button
- [ ] Search/filter for installed mods
