# UX-035: Customizable Keybindings

## Priority: T4 (Polish)
## Effort: Medium (3-5 days)
## Source: camera_controls_ux.md -- Section 14.4: Customizable Keybindings, master_architecture.md T4

## Description
`KeyBindings` resource with all keybindings as configurable fields. Settings UI for viewing and rebinding. Conflict detection and "Reset to Defaults" button. Saved to config file.

## Acceptance Criteria
- [ ] `KeyBindings` resource with all bindable actions
- [ ] Settings screen for viewing current bindings
- [ ] Click-to-rebind for each action
- [ ] Conflict detection with warning
- [ ] "Reset to Defaults" button
- [ ] Bindings saved to user config file
- [ ] Bindings loaded on startup
