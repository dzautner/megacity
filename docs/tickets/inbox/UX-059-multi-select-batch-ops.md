# UX-059: Multi-Select for Batch Operations

## Priority: T2 (Depth)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Selection Mechanics

## Description
Support selecting multiple buildings/roads for batch operations: batch bulldoze, batch upgrade, batch rezone. Works with box select (UX-011) and Ctrl+Click for adding to selection.

## Acceptance Criteria
- [ ] Ctrl+Click adds entity to selection
- [ ] Batch bulldoze: demolish all selected
- [ ] Batch upgrade: upgrade all selected roads
- [ ] Selection count shown in status bar
- [ ] Total cost for batch operation displayed
