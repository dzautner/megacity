# UX-038: Radial Tool Wheel (Controller)

## Priority: T4 (Polish)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 15: Controller Support

## Description
Radial menu for controller tool selection. Press X to show wheel, tilt stick to highlight category, release to select. Sub-wheel for items within category.

## Acceptance Criteria
- [ ] Hold X button shows radial wheel
- [ ] Stick direction selects category
- [ ] Release X confirms selection
- [ ] Sub-wheel for items within category
- [ ] Visual: circular menu with icons and labels
