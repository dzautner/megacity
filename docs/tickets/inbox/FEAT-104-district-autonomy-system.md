# FEAT-104: District Autonomy and Resource Specialization (Timberborn inspired)

**Category:** Feature / Governance
**Priority:** T4
**Source:** indie_competitors.md -- Timberborn

## Summary

Districts as semi-independent units with separate resource tracking. Controlled inter-district resource sharing via transfer points. New districts start as blank colonies. Strategic district placement for resource specialization. District-level budget tracking.

## Acceptance Criteria

- [ ] Per-district resource tracking
- [ ] Inter-district transfer controls
- [ ] District-level budget balance
- [ ] Specialization bonuses per district
