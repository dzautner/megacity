# FEAT-085: Transit Timetable and Frequency Management

**Category:** Feature / Transportation
**Priority:** T3
**Source:** community_wishlists.md -- Section 1.3

## Summary

Set departure frequencies/times for transit lines rather than just vehicle count. Express/local service on same rail line. Bus bunching prevention. Off-peak frequency reduction. Timetable coordination across lines.

## Acceptance Criteria

- [ ] Frequency setting per transit line
- [ ] Express/local service option
- [ ] Off-peak frequency reduction
- [ ] Timetable coordination at hubs
