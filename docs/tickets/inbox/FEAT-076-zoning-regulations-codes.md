# FEAT-076: Zoning Regulations and Building Codes

**Category:** Feature / Governance
**Priority:** T3
**Source:** community_wishlists.md -- Section 10.2, 20.1

## Summary

Height limits, setback requirements, FAR (floor area ratio) controls per district. Building codes: fire safety, earthquake resistance, accessibility. Environmental regulations: emission limits, EIA before development. Bureaucratic friction: permits, approvals, inspections.

## Acceptance Criteria

- [ ] Height limits settable per district
- [ ] FAR controls affecting building density
- [ ] Building codes affecting construction quality
- [ ] Environmental review for industrial projects
