# FEAT-096: Noise Pollution Model

**Category:** Feature / Environment
**Priority:** T2
**Source:** community_wishlists.md -- Section 6.1

## Summary

Noise levels from roads (proportional to traffic and speed), industry, airports, railways. Noise affects land value, health, happiness. Noise mitigation: sound walls, buffer zones, building insulation. Noise overlay visualization.

## Acceptance Criteria

- [ ] Noise calculated per source (road, industry, airport)
- [ ] Noise reduces land value and happiness
- [ ] Noise mitigation options
- [ ] Noise overlay visualization
