# UX-026: Road Parallel Snapping

## Priority: T2 (Depth)
## Effort: Small (1-2 days)
## Source: camera_controls_ux.md -- Section 12.6: Road Snapping

## Description
When drawing a road near an existing road, offer snap to maintain constant offset. Useful for one-way pairs or parallel service roads.

## Acceptance Criteria
- [ ] Parallel snap offered when cursor is near existing road
- [ ] Configurable offset distance
- [ ] Visual guide line showing parallel alignment
- [ ] Snap indicator visible during placement
