# FEAT-027: Crime Types and Prevention

**Category:** Feature / Social
**Priority:** T3
**Source:** community_wishlists.md -- Section 9.3

## Summary

Different crime categories (theft, vandalism, violent, organized). Crime prevention through lighting, community centers, employment (not just police). Prison/rehabilitation systems. Drug policy options affecting crime, health, tax revenue.

## Acceptance Criteria

- [ ] Multiple crime types with different causes
- [ ] Non-police crime prevention (lighting, employment, community)
- [ ] Prison vs rehabilitation options
- [ ] Crime policy tools per district
