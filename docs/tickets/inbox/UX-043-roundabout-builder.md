# UX-043: Roundabout Builder Tool

## Priority: T4 (Polish)
## Effort: Medium (3-5 days)
## Source: master_architecture.md T4

## Description
Automated roundabout creation: click center point, set radius, tool generates circular road with entry/exit connections to adjacent roads.

## Acceptance Criteria
- [ ] Click to set center point
- [ ] Drag to set radius
- [ ] Circular road segments generated automatically
- [ ] Entry/exit arms connect to nearby road endpoints
- [ ] Preview shows roundabout and connections
- [ ] Cost displayed during placement
