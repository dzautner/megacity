# UX-045: Actionable Advisor Tips

## Priority: T2 (Depth)
## Effort: Medium (2-3 days)
## Source: camera_controls_ux.md -- Section 13.7: Advisor System

## Description
Advisor messages should be non-intrusive (icon in notification area), actionable (include "Show Me" button to jump to location and/or activate tool), dismissible, and contextual.

## Acceptance Criteria
- [ ] Advisor icon in notification area (not modal popup)
- [ ] Each tip has "Show Location" button
- [ ] Each tip has "Dismiss" button (permanent per tip type)
- [ ] Tips only trigger when condition is active
- [ ] Example tips for: traffic congestion, fire coverage gap, budget deficit, zone demand
